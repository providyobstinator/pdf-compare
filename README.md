# PDF Difference Viewer

A simple, client-side website that compares two PDF files directly in the browser.

## Why this exists
Most people need to compare offer letters, policies, or documents but don't want to upload files to a server.
This tool works entirely on the client side using GitHub Pages.

## Features
- Compare two PDF files
- No backend, no uploads
- Runs fully in the browser
- Mobile-friendly
- Hosted on GitHub Pages

## Tech
- HTML
- CSS
- JavaScript
- PDF.js
